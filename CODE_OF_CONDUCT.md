Contributor Code of Conduct

Our Commitment

We are committed to fostering an open, welcoming, and respectful community for everyone, regardless of experience level, background, identity, or skill level.

NeoAfk is an educational open-source project. We encourage curiosity, learning, constructive discussions, and respectful collaboration.

Everyone participating in this project—including maintainers, contributors, and users—is expected to follow this Code of Conduct.

---

Expected Behavior

Examples of behavior that contribute to a positive community include:

- Being respectful and considerate.
- Welcoming newcomers.
- Providing constructive feedback.
- Being patient with questions.
- Accepting different viewpoints.
- Helping others learn.
- Focusing on technical discussions rather than personal disagreements.
- Giving credit where it is due.

---

Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment or discrimination.
- Personal attacks or insults.
- Trolling or intentionally disruptive behavior.
- Threats or intimidation.
- Publishing someone else's private information without permission.
- Spam or unrelated promotional content.
- Deliberately spreading misinformation about the project.

---

Technical Discussions

Disagreements about implementation are normal.

When discussing technical topics:

- Critique ideas, not people.
- Explain your reasoning.
- Be open to alternative solutions.
- Keep discussions professional and evidence-based.

The goal is to improve NeoAfk, not to "win" arguments.

---

AI-Assisted Contributions

NeoAfk welcomes contributors who use AI tools such as ChatGPT, Claude, GitHub Copilot, or similar assistants.

However:

- Review AI-generated code before submitting it.
- Verify correctness through testing.
- Ensure code follows the project's architecture and standards.
- Contributors remain responsible for the code they submit.

Using AI is encouraged as a productivity tool, not as a replacement for understanding the project.

---

Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

They may:

- Request changes to inappropriate behavior.
- Remove comments that violate these guidelines.
- Close discussions that become unproductive.
- Temporarily or permanently restrict participation for repeated violations.

Actions will be taken fairly and with the goal of maintaining a healthy community.

---

Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Community spaces related to NeoAfk
- Any interactions representing the NeoAfk project

---

Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please contact the project maintainer through GitHub.

Reports will be handled respectfully and, where possible, confidentially.

---

Attribution

This Code of Conduct is inspired by common open-source community standards, including the Contributor Covenant, and has been adapted to fit the goals and culture of the NeoAfk project.

---

Thank you for helping make NeoAfk a welcoming, educational, and collaborative open-source project.
